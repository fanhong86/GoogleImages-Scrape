NOTE: you need at least python 2.5 to run this.

A simple example to get started is here:
./mkdb.py googleImages bla -r 1 -n 1000 -o /tmp/mkdb_test

you may need to install some non standard python packages, and please refer to
./mkdb.py -h and the source for further options.
